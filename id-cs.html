Interior Design

Interior Design is a template website, designed and built to showcase projects.

My work at Interior Design

Responsive design and build

In today's digital world, responsive design and development play a pivotal role, ensuring seamless compatibility across all devices to improve user experience, accessibility, and navigational ease, also it significantly enhances visibility on search engines, amplifying online presence and engagement.

My main goal is to ensure that every page looks fantastic and functions smoothly on any device. I achieve this by using CSS tools like flexbox, grid, media queries, and fluid images. Whether someone is browsing on a computer, tablet, or phone, I want them to have a seamless experience. My aim is to create web pages that are user-friendly and visually appealing, regardless of how visitors access them.

The job

I created Interior Design as a template website to showcase various projects. Taking charge of both design and development, I crafted the logo and website layout using Figma. Building upon this foundation, I used HTML, CSS, JavaScript, and the UIkit front-end framework to bring the website to life.
